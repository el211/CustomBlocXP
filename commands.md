# Commands Reference

This document lists all the commands available in the CustomBlockXPPlugin and their usage.

## `/cxp reload`

- **Description:** Reloads the plugin configuration.
- **Usage:** `/cxp reload`
- **Permission:** `cxp.reload`

## `/cxp list <vanilla|custom>`

- **Description:** Lists all ores in the current chunk for either vanilla or custom blocks.
- **Usage:** `/cxp list vanilla` or `/cxp list custom`
- **Permission:** `cxp.list`

### Example Usage

```bash
/cxp list vanilla
